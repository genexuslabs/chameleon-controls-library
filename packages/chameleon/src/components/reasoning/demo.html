🧠 ch-reasoning - Interactive Demo

About this component: <ch-reasoning> is a Chameleon web component that displays AI reasoning steps in a collapsible accordion. It supports streaming content with a typewriter effect and pulse animations. The component is fully white-label and customizable via CSS parts and custom properties.

Example 1: Basic Usage (Default Styles)

A simple reasoning component with completed thought. Notice it has minimal default styling (white-label design).

Example 2: Custom Styling via CSS Parts

Same component, but styled using ::part() selectors. Check the CSS to see how header, section, and panel parts are styled.

Example 3: Streaming State with Pulse Animation

This example shows the streaming state. Notice the pulse animation on the header and the typewriter effect on the content.

Example 6: Custom CSS Properties

    
Customize the pulse animation via CSS custom properties:

  --ch-reasoning-pulse-duration: 0.8s;
  --ch-reasoning-pulse-opacity-min: 0.6;
  --ch-reasoning-pulse-opacity-max: 1;
}
