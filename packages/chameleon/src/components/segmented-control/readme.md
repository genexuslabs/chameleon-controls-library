# `ch-segmented-control-render`

<p>Segmented control is used to pick one choice from a linear set of closely related choices, and immediately apply that selection.</p>

<details open>
  <summary>
  
  ## Properties
  </summary>
  
### `exportParts:  string`

<p>Specifies the parts that are exported by the internal
segmented-control-item. This property is useful to override the exported
parts.</p>

**Attribute**: <code>exportparts</code>

**Default**: <code>SEGMENTED_CONTROL_EXPORT_PARTS</code>

---

### `itemCssClass:  string`

<p>A CSS class to set as the <code>ch-segmented-control-item</code> element class.
This default class is used for the items that don't have an explicit class.</p>

**Attribute**: <code>itemcssclass</code>

**Default**: <code>"segmented-control-item"</code>

---

### `model:  SegmentedControlModel | undefined`

<p>This property lets you define the items of the ch-segmented-control-render
control.</p>

**Default**: <code>undefined</code>

---

### `selectedId:  string | undefined`

<p>Specifies the ID of the selected item</p>

**Attribute**: <code>selectedid</code>

**Default**: <code>undefined</code>
</details>
